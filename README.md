# Project GrowthPath

Project GrowthPath is een leerplatform ontwikkeld om medewerkers binnen het innovatielab te helpen bij hun professionele groei met behulp van gamificatie-elementen. Het helpt medewekers met het vinden van leermateriaal die passen bij hun interesses en bestaande vaardigheden om te hun professionele zelf verder te ontwikkelen.
